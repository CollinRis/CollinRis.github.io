Alllergen Detector

OSU component made to detect Allergens when strings are inputed

As part of my Software II course at Ohio State, I created a custom component-based Allergen Detector to analyze ingredient labels and identify hidden allergens—something personal to me as someone with food allergies. Built using OSU’s component design principles, the system features a custom kernel, abstract class methods, design-by-contract specifications, and full JUnit testing. This project deepened my skills in modular software design and test-driven development, and I plan to expand its features further.
